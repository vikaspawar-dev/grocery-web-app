# Grocery Web App 🛒

A modern and responsive grocery shopping website built using React and Tailwind CSS.

## Features

* Responsive Design
* Product Categories
* Product Filtering
* Modern UI
* Reusable Components
* Favorite Product Icons
* React Router Navigation

## Categories

* Fruits & Vegetables
* Dairy & Eggs
* Meat & Seafood
* Organic Products

## Technologies Used

* React.js
* Tailwind CSS
* React Router DOM
* Vite

## Installation

```bash
npm install
npm run dev
```
